# My personal website

## Stack

- [Next.js](https://nextjs.org)
- [Tailwind](https://tailwindcss.com)
- Typescript
- [Vercel](https://vercel.com)

## Development

1. Clone this repository
   
   ```bash
   git clone https://github.com/saliven/saliven.com
   ```
  
2. Install dependencies
   
   ```bash
   yarn
   ```

3. Start development server
   
   ```bash
   yarn dev
   ```
